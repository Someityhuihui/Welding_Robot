# 基于机器视觉的焊接机器人自动路径规划系统

## 项目名称

WeldingRobot_Vision_PathPlanning
基于机器视觉的焊接机器人自动路径规划与运动控制系统

## 项目简介

本项目以六轴焊接机器人为研究对象，依托Rhino三维建模、ROS2+MoveIt2仿真平台，结合机器视觉焊缝检测、OMPL路径规划算法、正逆运动学求解、轨迹优化与运动控制，实现**工件建模→视觉焊缝识别→路点提取→自动路径规划→机器人仿真焊接**完整工程流程。
项目采用工业级标准目录架构，适用于课程设计、毕业设计、机器人仿真实训开发。

## 各文件夹详细说明

### 01_3D_Models

存放 Rhino 建模导出的 STL 模型：

- 机器人各连杆模型
- 焊接工件、工装、工作台、障碍物模型
   统一使用 STL 格式，供 RViz、MoveIt2 做可视化与碰撞检测。

### 02_ROS_URDF

存放机器人与环境描述文件：

- 机器人 URDF 模型文件
- 工件场景 URDF 配置
- MoveIt 配置文件、RViz 预设配置
   作用：为 ROS2、MoveIt2、RViz 提供机器人结构与环境模型基础。

### 03_Vision_Detection

机器视觉焊缝识别核心模块：

- 相机标定参数文件
- 图像采集与焊缝检测源码
- 点云处理、焊缝特征提取
- 输出焊缝三维坐标路点
   功能：自动从图像/点云识别焊缝位置，为路径规划提供原始焊接路点。

### 04_Path_Planning

焊接机器人路径规划模块：

- 直线/圆弧焊缝插补算法
- OMPL 全局无碰撞避障规划
- 路点管理、轨迹生成脚本
- 关节空间与笛卡尔空间轨迹处理
   接收视觉输出的焊缝路点，生成安全、平滑、符合焊接工艺的运动轨迹。

### 05_MoveIt_Control

基于 MoveIt2 的机器人运动控制模块：

- 正逆运动学求解接口
- 关节轨迹规划与下发
- 轨迹速度平滑、加减速优化
- 运动执行与状态监控
   功能：将路径规划生成的笛卡尔空间轨迹转换为机器人可执行的关节角度序列，实现精准、稳定、平滑的焊接运动。

### 06_Launch_Files

ROS2 系统启动文件集合，支持一键启动全流程：

- 总启动文件（视觉+规划+控制+可视化）
- 单独可视化启动
- 单独路径规划启动
- 单独视觉模块启动
   作用：简化开发调试流程，实现一条命令启动整个仿真系统。

### 07_Results_Video

仿真实验结果归档文件夹：

- RViz 仿真录屏
- 焊缝识别效果图
- 路径规划轨迹图
- 关节运动数据、位姿日志文件
   用于实验验证、成果展示、论文配图与项目复盘。

### 08_Documents

项目完整文档资料：

- 项目设计说明书
- 系统架构图、流程图
- 核心算法原理说明
- 毕业设计/课程设计论文
- 开发日志、问题总结
   用于项目交付、答辩展示、技术归档。

## 系统整体工作流程

1. 三维建模：使用 Rhino 建立机器人、工件、工作台模型，导出 STL。
2. 模型配置：编写 URDF，配置 MoveIt2 与 RViz 环境。
3. 视觉识别：采集图像/点云，自动提取焊缝三维坐标路点。
4. 路径规划：基于 OMPL 实现避障路径规划与轨迹插补。
5. 运动控制：通过 MoveIt2 完成逆解、轨迹优化、机器人驱动。
6. 仿真运行：在 RViz 中完成全自动焊接仿真并保存结果。
7. 文档归档：整理实验数据、截图、论文，完成项目结题。

## 运行环境

- 操作系统：Ubuntu 22.04 / WSL2
- ROS 版本：ROS2 Humble / Jazzy
- 仿真工具：RViz2、MoveIt2
- 建模软件：Rhino 7/8
- 视觉库：OpenCV 4.5+、PCL 1.12+
- 开发语言：Python 3.8+、C++17

## 项目功能亮点

1. 全流程自动化：从工件建模到仿真焊接无需人工干预。
2. 视觉引导焊接：支持图像/点云焊缝自动识别与定位。
3. 无碰撞路径规划：集成 OMPL 避障算法，保障运动安全。
4. 轨迹平滑优化：支持直线/圆弧插补、速度平滑，符合焊接工艺。
5. 工业级架构：目录规范、模块解耦，可直接扩展到真机部署。
