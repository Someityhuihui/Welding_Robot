PC2R 焊接轨迹规划系统 - 使用说明

一、系统简介

PC2R (Point Cloud to Robotic Welding) 是一个基于点云驱动的机器人焊接轨迹自动生成系统。系统读取多视角点云数据，自动完成点云拼接、焊缝边缘检测、焊接路径生成和焊枪姿态估计。

1.1 系统流程

text
多视角点云采集 → 点云拼接 → WSR提取 → 边缘检测 → 路径排序 → B样条拟合 → 姿态估计 → 轨迹输出

1.2 适用场景

  • 管道交叉焊缝的自动焊接路径规划

  • 大型复杂构件的机器人焊接

  • 多视角点云数据的自动拼接与重建

三、安装与编译

3.1 首次安装

bash
# 1. 克隆或复制项目到工作空间
cd ~/ros2_welding_ws
cp -r /path/to/trajectory_planning src/

# 2. 安装依赖
sudo apt update
sudo apt install -y ros-jazzy-pcl-conversions ros-jazzy-cv-bridge
sudo apt install -y libpcl-dev libeigen3-dev libopencv-dev

# 3. 编译
colcon build --packages-select trajectory_planning --cmake-args -DCMAKE_BUILD_TYPE=Release

# 4. 设置环境
source install/setup.bash

3.2 重新编译（修改代码后）

bash
cd ~/ros2_welding_ws

# 清理（可选）
rm -rf build/trajectory_planning install/trajectory_planning

# 编译
colcon build --packages-select trajectory_planning --cmake-args -DCMAKE_BUILD_TYPE=Release

# 刷新环境

系统需要5帧多视角点云数据，目录结构如下：

text
dataset_folder/
├── 1_frame.pcd          # 第1帧点云
├── 1_frame.txt          # 第1帧相机位姿
├── 2_frame.pcd
├── 2_frame.txt
├── 3_frame.pcd
├── 3_frame.txt
├── 4_frame.pcd
├── 4_frame.txt
├── 5_frame.pcd
└── 5_frame.txt

4.2 相机位姿文件格式 (txt)

每帧的位姿文件包含一行三列数据，Tab分隔：

text
x坐标    y坐标    z坐标
0.756   0.200   0.015

4.3 配置数据集路径

编辑 include/welding_config.hpp：

cpp
struct DataPathConfig {
    std::string dataset_folder = "/path/to/your/dataset/";
    std::string output_folder = "/path/to/output/";
};

或在运行时通过ROS话题设置：

bash
ros2 topic pub /pointcloud_storageFolder std_msgs/msg/String "data: '/path/to/dataset/'"

七、参数配置

7.1 配置文件位置

include/welding_config.hpp

7.2 主要参数说明

边缘检测参数

cpp
struct EdgeDetection {
    float search_radius = 0.025f;     // 搜索半径(m)，增大→更多边缘点
    float epsilon = 1.2f;             // 响应阈值，降低→更多边缘点
    float intensity_threshold = 0.002f; // 强度阈值，降低→更多边缘点
    int min_neighbors = 5;             // 最小邻居数，降低→更宽松
};

降采样参数

cpp
struct AdaptiveDownsample {
    size_t target_points_min = 30000;   // 目标最小点数
    size_t target_points_max = 50000;   // 目标最大点数
    float voxel_size_min = 0.001f;      // 最小体素(m)
    float voxel_size_max = 0.008f;      // 最大体素(m)
};

WSR提取参数

cpp
struct WSRExtraction {
    float cylinder_radius = 0.16f;      // 圆柱滤波半径(m)
};

7.3 参数调整建议

问题调整方法
边缘点太少增大search_radius，降低epsilon
边缘点太多（噪声）减小search_radius，提高epsilon
处理太慢减小target_points_max，减小cylinder_radius
路径不平滑调整bspline.sampling_rate

十、性能参考

边缘点200-800输出

十一、注意事项

  1. 首次运行较慢：边缘检测需要30-60秒，请耐心等待

  2. 数据集路径：确保路径末尾有 /

  3. WSL用户：需要设置 export LIBGL_ALWAYS_SOFTWARE=1

  4. 内存占用：处理大型点云时可能占用2-4GB内存

  5. 参数调整：修改参数后需要重新编译
